# ETF/个股 做T交易记录

一个**纯前端**（Next.js + React + TypeScript）的个人交易记录小工具，用于记录多个 ETF/个股的做 T 交易，并在本地浏览器中进行统计展示。

> 说明：这是**交易记录/复盘工具**，不接入任何行情数据，不是交易系统。

## 主要功能

- **多标的管理**：创建多个 ETF/个股条目（含费率/滑点/最小交易单位等设置）。
- **交易记录 CRUD**
  - 新增交易
  - 编辑交易（行内编辑）
  - 删除交易（带确认）
- **自动计算（基于交易序列）**
  - 逐笔计算：成交额、费用、净现金流、持仓前后、均价前后、单笔盈亏、累计已实现盈亏
  - ETF 汇总：当前持仓、当前均价、持仓市值、已实现盈亏、今日 T 统计
  - 今日汇总：今日卖出笔数、今日已实现盈亏（不区分是否 T）
- **数据持久化**：所有原始数据存储在浏览器 `localStorage`。
- **备份与恢复**：导出/导入 JSON（覆盖式导入，带确认）。

## 本地开发

### 环境要求

- Node.js 18+（建议 18 或 20）
- npm

### 安装与启动

```bash
npm install
npm run dev
```

启动后访问：

- `http://localhost:3000`

### 构建与运行（可选）

```bash
npm run build
npm start
```

## 数据说明（重要）

- 本项目仅在浏览器本地运行，数据保存在 `localStorage`。
- **不会上传任何数据**，也**不连接任何行情**。
- 建议定期使用首页的「导出数据」功能备份。
